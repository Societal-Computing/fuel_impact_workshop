# Did fuel-tax cuts get Europe driving again? — a hands-on workshop

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/Societal-Computing/fuel_impact_workshop/blob/main/workshop.ipynb)

A short, hands-on taste of **data-driven research** for students who can code a
little but have never done data analysis. You'll use **real data** behind a
scientific paper to answer a real question: when fuel prices spiked across Europe
in 2026 and sixteen governments cut fuel taxes, did the cuts change how people drove?

## ▶️ Start here
Click the **Open in Colab** badge above, then **Runtime → Run all**. That's it —
nothing to install, and the data loads itself. *(You'll need a free Google account.)*

Prefer your own machine? See **[HOW_TO_RUN.md](HOW_TO_RUN.md)**.

## What you'll do
1. **Collect** data — a live weather API call and one real Google "live busyness" record.
2. **Plot** how an entire continent's driving changed over six weeks (a "V-shape").
3. **Validate** the signal against independent motorway car-counts — do two totally different systems agree?
4. **Catch a trap** — three countries that cut *no* tax show the same pattern. So did the cuts cause it?

A few cells are marked `# TODO` for you to finish — that's the fun part.
